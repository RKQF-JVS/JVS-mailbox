# JVS-mailbox

#### 介绍
企业邮筒，基于B/S模式的邮件客户端，采用JVS的统一用户体系，作为JVS的协同办公的应用之一。


#### 项目标签
低代码、微服务、支持SaaS、私有化部署、Web邮件客户端


#### 软件架构
软件架构说明，JVS-mailbox是作为JVS基础框架之上的原生应用，使用相同的用户体系，可以通过企微、钉钉、公众号、用户密码等多种方式登录邮箱。
